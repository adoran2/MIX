Current Stuff - ASSEMBLER
-------------------------

Extend TextField so that all input is converted to upper-case.  Very important
as we don't want the user to input lower-case symbols - these are to be used
internally by the assembler.

With LABEL EQU EXPRESSION we need to set up a LIST of labels with the following
data: label
      line number (in case of errors)
      expression (for parsing before being entered into the symTable)
Now, when we finish pass1, we need to scan this list iteratively.  If we can't
resolve one of the expressions, we move on to the next.  On each pass, the list
should be smaller than it was before, until it is zero and all labels are in the
symTable.  If, after two consecutive iterations, we have the same-sized list,
there is an undefined symbol error at one of the lines.  Can list them all!
